docs(cloud): CODEX_AUTH_JSON 投入手順を compact JSON + 環境変数欄に訂正

2026-06-28 cloud session 実機検証で 2 点判明、ドキュメント訂正。

(1) cloud Claude Code on the web には secret 専用欄が無い(anthropics/claude-code#32733
    feature request 段階、未実装)── 環境変数欄(plaintext + collaborator 共有可視)が
    唯一の投入経路。UI 警告「機密情報や認証情報は追加しないでください」を無視して投入、
    prolegomena cloud environment の collaborator は人見単独前提(人見承認済)。
    collaborator add 時は cloud Codex 主経路退役。

(2) auth.json を Get-Content -Raw で取った整形済 multi-line JSON を環境変数欄に raw
    で貼ると、保存時 .env 形式パーサが改行で行を切り value 後半が silently truncate
    される(2026-06-28 検証で再現、value が消える現象として顕在化)。必ず
    ConvertTo-Json -Compress で 1 行 compact 化してから貼付。
