feat(cloud): Codex auth portable で cloud session の Codex 主経路復活

OpenAI 公式 portable auth flow を採用し、local `~/.codex/auth.json` 全文を `CODEX_AUTH_JSON` env var(cloud UI secret 欄)経由で cloud に持ち込む経路を整備。`CODEX_AUTH_JSON` 有り時は cloud でも Codex Bash 直叩きを実装主経路として復活(モデルレベル物理分離維持)、無し時は従来通り Claude 内サブエージェントへフォールバック。
